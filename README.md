>> Amazon Product Analytics

>> Project Overview
This project focuses on cleaning raw Amazon product data using SQL and building a Power BI dashboard to analyze pricing, discounts, and ratings.

>> Tools Used
- SQL (SQLite)
- Power BI
- Excel

>> Data Preparation Steps
- Created surrogate primary key
- Converted text price fields into numeric columns
- Calculated discount percentage
- Built analytics view for reporting

>> Business Questions Answered
- Which products offer the highest discounts?
- What is the average rating across categories?
- How do pricing and discounts compare?

>> Dashboard Features
- Average Rating (KPI)
- Average Discount % (KPI)
- Top 10 Products by Discount
- Average Rating by Category
- Product Price & Rating Table

- amazon_analytics.csv
